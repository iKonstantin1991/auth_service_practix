## Описание
Auth сервис для онлайн кинотеатра practix 

## Команды для запуска приложения

- создать `./auth-service/.env` в соответствии с `./auth-service/.env.template`
- выполнить:

```
cd ./infra
docker-compose --project-name auth-api up -d
```
